# HACS Vision 后期规划

> 日期：2026-06-07
> 项目：HACS Vision — HACS 增强前端面板

---

## 项目定位

HACS Vision 是 HACS 的最佳替代前端，目前已有功能：
- 商店浏览（卡片/列表视图、搜索、分类/状态筛选、排序、分页、分组）
- 服务端收藏持久化
- 更新管理（多选批量更新、Changelog 弹窗、实时状态轮询）
- 管理页面（自定义仓库增删、归档/重命名仓库处理、备份导入导出、依赖检查）
- 一键替换重命名仓库
- GitHub API 限流解决（复用 HACS Token）
- 26 个后端 API 端点
- 搜索支持 manifest_name + 完整 GitHub URL

---

## 第一梯队 — 轻量高价值

### 1. 添加集成入口（已确认方案）
- 在仓库卡片右下角操作区增加「添加集成」按钮
- **触发展示逻辑**：
  | 仓库状态 | 显示按钮 |
  |----------|---------|
  | 未安装 | [安装] |
  | 已安装 + 是集成 | [移除] [添加集成] |
  | 已安装 + 非集成 | [移除] |
- 通过 `hass.navigate('/config/integrations')` 内部跳转，不新开窗口
- 尝试带 `?search=集成名` 参数预填充 HA 原生搜索框
- 用户流程闭环：逛商店 → 安装 → 添加集成，一条链路搞定

### 2. 搜索增强
- 支持完整 GitHub URL 搜索（如 `https://github.com/owner/repo`）
- 支持 `manifest_name`（HA 显示名，如 "HACS Vision"）搜索
- 改动极小，后端搜索条件增加两行

### 3. 更新推送通知
- 通过 HA 的 `persistent_notification` 发通知
- 可选配置手机推送（`notify.mobile_app`）
- 仅在有可用更新时推送，避免骚扰

### 2. 批量操作扩展
- 批量安装（商店多选 → 一键安装选中）
- 批量移除（已安装多选 → 一键移除）
- 批量收藏/取消收藏

### 3. 快捷安装
- 复制 GitHub URL 时自动检测并弹出安装提示
- 在任何界面通过浮动入口一键安装
- 自动识别分类（从 HACS 仓库信息推断）

### 4. 配置页面
- 刷新间隔设置
- 默认视图设置
- 通知开关
- 语言偏好

## 第二梯队 — 中投入高价值

### 5. 集成健康面板
- 显示已安装的集成是否正常工作
- 报错数量、最后在线时间
- 与 HA `system_log` 联动

### 6. 仓库备注
- 给已安装仓库添加自定义备注
- 存储在 `.storage/hacs_vision_notes.json`

### 7. 更新预演
- 更新前显示变更文件列表
- 确认后再执行更新

### 8. 分类标签系统
- 用户自定义标签分组
- 跨分类的仓库组织能力
