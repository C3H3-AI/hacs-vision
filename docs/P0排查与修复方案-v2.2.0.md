## 3. P0-2 根因分析：Backup 数据源错误

### 3.1 数据对比

| 端点 | 数据来源 | 返回数量 | 正确？ |
|------|---------|---------|--------|
| `/installed` | `operator.get_installed_list()` | **84** | ✅ |
| `/config/custom` | `operator.get_all_repos_from_hacs()` | **24** | ✅ |
| `/backup/export` | `data.get_installed_repositories()` | **3144** | ❌ |
| `/config` | `data.get_config()` | 无 custom_repositories | ❌ |

### 3.2 根因

`backup.export()` 使用 `HACSData` 文件读取路径：

```python
# backup.py L13 — 坏路径：hacs.data 包含 ALL repos，不区分 installed
installed = await self.data.get_installed_repositories()

# backup.py L15 — 坏路径：hacs.hacs 不包含 custom_repositories key
config = await self.data.get_config()
```

正确的数据源于 `HACSOperator`（HACS 内存对象）：
- `operator.get_installed_list()` → 过滤 `repo.data.installed == True`
- `operator.get_all_repos_from_hacs()` → 通过 `repo.data.custom` 识别自定义仓库

### 3.3 修复方案

**方案 A（推荐）：Backup 使用 operator 数据源**

改动文件：`backup.py`, `__init__.py`

```python
# backup.py — 新增 operator 参数 + 重写 export()
class BackupManager:
    def __init__(self, hass, shared_data=None, operator=None):
        self.data = shared_data or HACSData(hass)
        self.operator = operator  # NEW
        self.hass = hass

    async def export(self) -> dict:
        installed = []
        custom = []
        if self.operator and self.operator.available:
            installed = self.operator.get_installed_list()
            all_repos = self.operator.get_all_repos_from_hacs()
            custom = [r for r in all_repos if r.get("custom")]
        else:
            # Fallback: file-based data
            installed = await self.data.get_installed_repositories()
            config = await self.data.get_config()
            custom = config.get("custom_repositories", [])
        
        return {
            "exported_at": datetime.now().isoformat(),
            "version": "1.0",
            "installed": [{
                "full_name": r.get("full_name", ""),
                "category": r.get("category", "integration"),
                "installed_version": r.get("installed_version", ""),
            } for r in installed],
            "custom_repositories": [{
                "repository": r.get("full_name", ""),
                "category": r.get("category", "integration"),
            } for r in custom],
        }
```

```python
# __init__.py L35 — 传入 operator
backup = BackupManager(hass, shared_data=shared_data, operator=operator)
```

**方案 B（加固）：`get_installed_repositories()` 收紧过滤**

改动文件：`hacs_data.py` L128-134

```python
# Before
for cat_repos in cat_dict.values():
    if isinstance(cat_repos, list):
        installed.extend(cat_repos)

# After
for cat_repos in cat_dict.values():
    if isinstance(cat_repos, list):
        for repo in cat_repos:
            iv = repo.get("installed_version") if isinstance(repo, dict) else None
            if iv:  # truthy: catches "", None, missing key
                installed.append(repo)
```

## 4. 附加修复：`_get_translations` 阻塞 I/O

**位置**：`api.py:997`

**HA 日志**：
```
Detected blocking call to open inside the event loop
by custom integration 'hacs_vision' at api.py, line 960
```

**修复**：

```python
# api.py L997 — Before
with open(trans_path, "r", encoding="utf-8") as f:
    data = json.load(f)

# After
content = await self.hass.async_add_executor_job(
    self._read_file_sync, trans_path
)
if content is not None:
    data = json.loads(content)
```

需要新增 helper：
```python
def _read_file_sync(self, path: str) -> str | None:
    try:
        with open(path, "r", encoding="utf-8") as f:
            return f.read()
    except FileNotFoundError:
        return None
```
