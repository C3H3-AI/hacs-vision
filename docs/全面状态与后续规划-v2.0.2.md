### 二、配置弹窗与集成添加流程评估

#### 流程完整性评估

```
用户路径 1: 仓库卡片 → 点击齿轮图标 → 已配置 → 打开 Options Flow
用户路径 2: 仓库卡片 → 点击齿轮图标 → 未配置 → 打开 Config Flow (新集成)
用户路径 3: 设置页 → 添加 HA 集成 → 搜索选择器 → 打开 Config Flow
用户路径 4: 设置页 → 添加 HA 集成 → 多实例选择 → 打开 Options Flow
```

所有 4 条路径均可正常走通。

#### 配置弹窗 UI 评分

| 维度 | 评分 | 说明 |
|---|---|---|
| **信息完整性** | ✅ 9/10 | 所有字段/选项/错误/结果均有翻译覆盖 |
| **交互流程** | ✅ 9/10 | form/menu/external 全支持，30s 超时兜底 |
| **错误处理** | ✅ 8/10 | 网络错误、超时、认证失败均有中文提示 |
| **结果展示** | ✅ 9/10 | 4 种结果状态（成功/中止/失败/外部认证）均有图标+文字 |
| **移动端适配** | ✅ 8/10 | 响应式 dialog，slideUp 动画 |
| **无障碍** | ❌ 5/10 | 无 aria-*，无键盘焦点管理 |

**存在的细节问题**:
- 配置结果页的 SVG 图标使用 `color` 而非 `fill`，部分 HA 主题下可能显示不正常
- 无 `aria-label` 或 `aria-describedby` 无障碍属性
- 外部认证打开新窗口后，焦点未自动返回
